# Specification

## Summary
**Goal:** Build a minimal, single-page developer portfolio website with a clean light/dark design and smooth-scroll navigation.

**Planned changes:**
- Sticky top navigation bar with anchor links to all five sections and a dark mode toggle; collapses on mobile
- Hero section: centered name placeholder, "Developer" tagline, and one-liner intro; plain white background, no imagery
- About section: short placeholder bio paragraph with clean typography
- Skills section: minimal pill/tag grid of 7+ placeholder skills (HTML, CSS, JavaScript, React, Node.js, TypeScript, Git)
- Projects section: three placeholder project cards each with title, description, GitHub link, and Live link; subtle border/shadow and hover lift effect; responsive stacking on mobile
- Contact section: placeholder email address with GitHub and LinkedIn links; no form
- Design system: Inter/system-ui font, near-black text on white/off-white background, one muted accent color (teal or slate), generous vertical spacing, subtle hover effects only, clean dark mode palette inversion

**User-visible outcome:** A fully static, single-page portfolio site the user can open in a browser, scroll through all sections, toggle dark mode, and use as a personal developer portfolio template.
